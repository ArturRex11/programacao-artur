# Artur Rex 11

Portfólio simples e elegante em HTML, CSS e JavaScript para apresentar links e redes sociais.

## Sobre

Este projeto é uma landing page pessoal que mostra um perfil com foto, links para projetos e redes sociais, e um botão para alternar entre modo escuro e modo claro.

## Recursos

- Layout responsivo para mobile
- Tema escuro / claro com transição visual
- Links para GitHub, Instagram, YouTube e Twitch
- Uso de ícones do Ionicons
- Design limpo com bordas em vidro fosco e efeitos de hover

## Arquivos

- `index.html` - estrutura da página
- `style.css` - estilos e temas
- `script.js` - controle do botão de alternância de tema
- `assets/` - imagens e recursos visuais

## Como usar

1. Abra `index.html` no navegador.
2. Clique no botão do switch para alternar entre modo claro e escuro.
3. Use os links para navegar para os perfis e projetos.

## Personalização

- Altere `#profile img` em `index.html` para usar sua própria foto.
- Modifique as cores e fundos em `style.css` para ajustar o visual.
- Adicione ou remova links na lista de navegação conforme necessário.
